## 问题1: 节点评估失败 (根本原因)

### 现象
- 日志显示 "Evaluated 0 nodes"
- 节点列表一直显示"评估中..."

### 根本原因 (已确认)
Mihomo配置中**没有** `🚀 节点选择` 这个组名。实际可用的selector组是：
- `AnyTLS` (37个节点)
- `GLOBAL` (49个节点)  
- `Tuic`
- `狗狗加速.com`
- `🔥ChatGPT`

原代码硬编码了不存在的组名，导致API返回404。

### 修复方案 (已实施)
```python
# 旧代码 - 硬编码
group_name = "🚀 节点选择"  # 不存在！

# 新代码 - 自动检测
selector = MihomoAPI._find_best_selector()  # 返回 "GLOBAL"
```
